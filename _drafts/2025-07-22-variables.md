In C/C++, in the easiest way, variables and constants are objects which may have information that we can/must put inside. Imagine them like a 1 dollar bill that may possess a value. The dollar bill is the variable.

int integer;

Here, we declared a variable. int means that it will be an integer type, so it's gonna be a variable that will retain integer numbers. But for now, integer is a undefined variable because we did not assign it a value.

To assign a value into it there are 2 ways:

int integer = 30;
integer = 50; 

Notice that we are creating a variable now. So we can't declare it again later.
